## ✨ 核心能力

- **内页的概念**：新闻站详情页、视频站的具体播放页、资源内页、统一成为内页。
- **三种采集模式**，适配任何网页结构，可应对不同场景：

  | 模式 | 适用场景 | 说明 |
  |------|----------|------|
  | **翻页采集** | 带分类/列表栏目的网站 | 自动遍历多页，提取结构化数据 |
  | **内页采集** | 无分页，但ID递增或内页含内页的网站 | 适用于资源详情页、文章内页等 |
  | **监控数据** | 固定页面变化监测 | 设定条件触发通知，如商品降价、内容更新 |

---

## 🚀 使用流程

1. **填写目标网址与采集间隔**
2. **通过「保存器」定义要采集的字段名**（列名）
3. **提交任务，云端自动执行则无需下载软件，也可以下载软件本地采集**
4. **导出或分发采集结果**

## 🧠 人工智能创新

### 基于大语言模型（LLM）的智能采集
- **代码模式**：支持自定义脚本，自由扩展，满足高级需求。
